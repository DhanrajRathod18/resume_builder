# Resume Builder

A dynamic and modern Resume Builder built with React and Vite.

## Links

- **Live Site**: [Open Resume Builder](https://DhanrajRathod18.github.io/resume_builder/)
- **Repository**: [GitHub Project](https://github.com/DhanrajRathod18/resume_builder)

## Features
- Real-time live preview.
- PDF generation and printing.
- Modern, clean, and responsive UI.

## Getting Started

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/DhanrajRathod18/resume_builder.git
   ```
2. Navigate to the project directory:
   ```bash
   cd resume_builder
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
